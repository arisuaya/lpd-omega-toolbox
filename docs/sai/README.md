从Google Play提取的部分软件采用的是.xapk或者是.apks格式的安装包，安卓设备无法直接进行安装，故我们需要使用SAI安装器来进行安装
接下来我们来看看如何使用SAI来安装程序  
tips.鸿蒙Harmony OS可以直接点击安装.xapk格式的安装包

## **使用方法**  
**1. 下载SAI的安装包并安装**  
   <img src="https://amazonaws.urauto.ltd/statics/sai-1.jpg" width="40%">  
**2. 打开SAI，如果你是MIUI用户此处会弹出此页面，请点击“开发者选项”按钮，进入开发者选项关闭MIUI优化**  
   <img src="https://amazonaws.urauto.ltd/statics/sai-2.jpg" width="40%">
   <img src="https://amazonaws.urauto.ltd/statics/sai-3.jpg" width="40%">  
**3. 回来后点继续，进入SAI主界面，点击安装APK，选择内置**   
   <img src="https://amazonaws.urauto.ltd/statics/sai-4.jpg" width="40%">
   <img src="https://amazonaws.urauto.ltd/statics/sai-5.jpg" width="40%">  
**4. 找到你下载.Apks文件的路径，确定，安装**  
   <img src="https://amazonaws.urauto.ltd/statics/sai-7.jpg" width="40%">
   <img src="https://amazonaws.urauto.ltd/statics/sai-8.jpg" width="40%">  
**5. 剩下的步骤与正常的安装一致，安装完毕后即可打开软件**  
   <img src="https://amazonaws.urauto.ltd/statics/sai-10.jpg" width="40%">  
**6. MIUI优化记得在安装后开回来，不然系统主题会变为原生样式**  
   <img src="https://amazonaws.urauto.ltd/statics/sai-3.jpg" width="40%">
